Knitting

  • Using interlocking loops to create fabric
  • Descriptor of the fabric during creation
  • Hobby of mine since 2001

Knitting used to be considered an activity for older ladies. Now it's more widespread and knitters can be found in every demographic. It can be as easy or difficult, as affordable or pricey as you want.


Sometimes I'd pick a fairly easy project and knit while walking. When that walking is done at San Diego Comic-Com, it can make for interesting photo opportunities.


Knitter's Review is a great resource for knitters. The layout is well-defined and you can explore different areas from the top nav area.


"Properly practiced, knitting soothes the troubled spirit, and it doesn't hurt the untroubled spirit, either." -- Elizabeth Zimmerman
